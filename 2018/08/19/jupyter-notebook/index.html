Jupyter Notebook 安装及入门使用

2018-08-19 / by Tom

这里记录一下jupyter notebook 的安装以及初级入门使用过程。参考Jupyter Notebook介绍、安装及使用教程

以下均在Windows上面进行，关于Mac/Linux可以看参考的文章。

安装

  • 前提：电脑已经安装Python
  • 使用pip命令安装：pip3 install jupyter

启动

  • 在终端(Windows: win+R, 输入cmd)中输入以下命令：jupyter notebook
  • 之后会自动打开默认浏览器，打开8888端口，如果打开多个，端口号从8889依次递增，使用过程中不要关掉终端；
  • 第一次打开会发现已经有很多文件夹，是自己的电脑的文件，希望更改文件存放地址，可以根据以下步骤来给自己一个清爽的界面：
    • 在自己想要存放jupyter notebook文件的位置创建目录（右上角有个new-folder),接下来去配置文件中修改默认路径（如下）；
    • 在终端输入命令jupyter notebook --generate-config，查看配置文件路径；
    • 根据上面查到的路径，打开配置文件jupyter_notebook_config.py(这边我用的是VS code,其他的IDE都可以)；
    • 在配置文件中查找c.NotebookApp.notebook_dir,把刚刚自己新建的文件路径填写进去，然后取消注释并保存，然后再在终端中运行jupyter notebook，就会发现打开的就是自己后来建的文件夹；
    • 说明：根据参考的那篇文章，是复制浏览器地址栏的路径，但是我这边会报错，于是我直接把文件具体路径输入进去，(我的是c.NotebookApp.notebook_dir = 'C:/Users/ye/Desktop/for_jupyter_notebook')；

  • 上述命令就将jupyter notebook与conda环境关联起来了；

Markdown

  • 安装：终端输入命令conda install -c conda-forge jupyter_contrib_nbextensions

入门使用

code一下

  • 新建ipynb文件：在jupyter notebook的主界面file下，右侧new一个Python即可；
  • 重命名文件：可以在主界面选中文件点Rename，也可以在文件打开的页面，直接点击名字就会弹出重命名的窗口；
  • 在cell里面输入代码，按ctrl+Enter，执行当前cell代码；

markdown

  • markdown可以用于代码的注释；点击菜单栏的像目录一样的图标，里面就会出现目录；
  • 插入cell后可以在菜单栏手动选择为Markdown，也可以在命令模式下使用快捷键M;
  • 语法就是Markdown语法；

matplotlib

  • 绘图时，运行后只显示一个内存地址，而没有图，解决方法是使用magic function： %
  • 使用语句： %matplotlib inline

快捷键

  • 快捷键的查询可以在点击help–keyboard shortcuts
  • 下面记录几个常用快捷键,分为命令模式和编辑模式：
    • 命令模式按esc（或者鼠标直接点击外面部分）,当前选中cell前面会是蓝色竖线；
    • 编辑模式按Enter（或者鼠标点击一下cell），当前选中cell前面是绿色竖线；
keyboard shortcuts function Mode
ctrl+enter 只执行当前cell
shift+enter 执行当前cell并跳到下一个cell，如果下面没有cell则新建
alt+enter 执行当前cell并在当前cell后面新建一个cell
D+D 删除当前选中cell 命令模式
M 切换为MarkDown 命令模式
Y 切换为code 命令模式
F 查找 命令模式
Z 撤销删除cell操作 命令模式
1(2,3,4,5,6) 直接把当前选中cell第一行切换为1(2,3,4,5,6)级标题 命令模式

进阶使用

魔法函数

  • 符号说明：
    • %： 行魔法函数，只对本行代码生效；
    • %%：Cell魔法函数，对整个cell生效，放在cell首行；
    • %Ismagic: 列出所有的魔法函数；
    • %magic: 查看各个魔法函数的说明
    • ?：后面加上魔法函数 名称，查看该函数说明；
  • 常用魔法函数：
magic function function explanation
%pwd 查看当前目录
%ls 列出当前目录的文件
%run 运行调用外部Python脚本 后面跟Python脚本的绝对路径
%timeit 单行语句的执行时间
%%timeit 整个cell内语句的执行时间
%matplotlib inline 显示matplotlib生成的图形 用在画图之前
%reset 清除所有变量
%pdb 调试程序
%who 查看所有全局变量的名称，若给定类型参数，只返回该类型的变量列表
%whos 显示所有全局变量的名称、类型、值/信息
%xmode Plain 设置为当异常发生时只展示简单的异常信息
%xmode Verbose 设置为当异常发生时只展示详细的异常信息
%debug bug调试，输入quit退出调试
%env 列出全部环境变量
